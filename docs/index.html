Introduction

Ce tutoriel guide les étudiants pas à pas pour comprendre le développement d'une application web complète avec frontend HTML/CSS/JS et backend Node.js/Express.

Objectifs pédagogiques

  • Comprendre l'architecture client-serveur 2 tiers
  • Manipuler le DOM et Fetch API
  • Créer et tester des routes Express
  • Documenter et présenter un projet web complet

Frontend – HTML/CSS/JS

Le frontend permet de :

  • Afficher les événements
  • Ajouter de nouveaux événements via un formulaire
  • Communiquer avec le backend via Fetch API

// Exemple JS pour récupérer les événements
fetch('/events')
  .then(resp => resp.json())
  .then(data => console.log(data));

Backend – Node.js & Express

Le backend est un serveur Express exposant des routes pour :

  • GET /events → récupérer tous les événements
  • POST /events → ajouter un nouvel événement

// Exemple serveur Express
const express = require('express');
const app = express();
app.use(express.json());
app.use("/events", eventsRoutes);
app.listen(3000, () => console.log('Serveur démarré'));

Conclusion

En suivant ce tutoriel, vous pouvez comprendre comment le frontend et le backend communiquent, manipuler le DOM, tester des routes avec Postman/Thunder Client et créer un site fonctionnel complet.
